## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`GOOGLE_ID`

`GOOGLE_SECRET`

`NEXT_PUBLIC_SUPABASE_URL`

`NEXT_PUBLIC_SUPABASE_ANON_KEY`

`SUPABASE_SERVICE_ROLE_KEY`

`NEXTAUTH_SECRET`

`NEXTAUTH_URL`

`DATABASE_URL`

`NODE_ENV`
